#skeleton
-----
##简介
#####skeleton 是一个 JavaEE 项目快速开发脚手架，集成了常用的 JavaEE 框架和前端开发框架，拿来即用，省去繁琐的配置，节约下来的时候拿去陪女朋友吧，哈哈 :heart_eyes:  

---
##技术选择

###后端
1. 基于 Spring 管理 Bean
2. Dao 层使用 [Hibernate](https://github.com/veryyoung/skeleton)/[Mybatis](https://github.com/veryyoung/skeleton/tree/mybatis)
3. MVC 使用 SpringMVC
4. 验证框架 hibernate-validator
5. 日志管理 slf4j+logback
6. 数据源 druid
7. jcaptcha 验证码
8. jsp+servlet 

###前端
1. Sea.js 组织 js 代码
2. jquery+js
3. jquery-ui-bootstrap 界面框架
4. font-wesome 字体/图标框架


方便的前后端验证，自定义 tag，基于注解的登陆认证等。可扩展性良好，设计精巧，使用简单。大家快来用吧！:blush:
